## JS

### 자료형  
--var : 중복선언 O, 재할당 O  
--let : 중복선언 X, 재할당 O  
-- const : 중복선언 X, 재할당 X  
  
### 연산자  
  
#### x++ 과 ++x 의 차이점

x++ //선 대입 후 연산  
++x //선 연산 후 대입  

#### == 과 ===의 차이점  
  
== 은 변수의 타입 고려 X 값만 같으면 O  
=== 은 변수의 타입 + 값까지 같아야 O
